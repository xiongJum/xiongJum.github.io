  • 参考/引用：鸟哥的Linux私房菜-第十一章

  • 命令编修能力

  • 命令与文件补全能力[tab]

    • [tab]接在命令之后为[命令补全]
    • [tab] 接在命令-文件之后为[文件补齐]
    • 连续按两次[tab]显示所有可运行的命令
    • c[tab] [tab]显示以c开头的所有可运行的命令

  • 命令别名配置功能（alias）

    • alias lm='ls -al'
# lm 等同于命令 ls -al
# 查看目前命名的所有别名（alias） 
alias

  • 命令的下达

    • 如 “\ [Enter]” 使 [Enter]不再具有[开始运行]的功能，具有转义的作用，并将命令在下一行继续输入
